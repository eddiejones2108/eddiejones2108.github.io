Linear Logic

Suppose apples cost £1; if you have £1, you can have an apple. In fact we might be tempted to write £1 ⇒ 🍏, but from this we can derive £1 ⇒ 🍏 ⋀ 🍏. There seems to be some sort of buy one get one free offer on?

Linear logic acts as an accountant for greengroceers by considering resources rather than true, but unlike truth, resources can be used up. The proposition £1 ⊸ 🍏 specifically states that the pound is consumed in the production of the apple. It does not follow, therefore, that £1 ⊸ 🍏 ⊗ 🍏.

Here the ⊗ symbol acts somewhat like classical conjunction, but clarifies that we are actually talking about two separate apples. So £1 ⊗ £1 means something like £2.

Linear Algebra

Linear algebra on the other hand is to the nearest approximation the study space and linear transformations. The type of space were intrested in can be abstract, but is required to behave somewhat like Euclidean space, so your intuition should suffice. Let's unpack this notion of transformation however:

A linear transformation is a function that preserves the origin. That is the zero point is mapped to the zero point. Futhermore, any straight lines in the input space must also be straight lines in the output space. That is it! This class of functions is surpisingly common as it include rotations and scaling in any combination.
