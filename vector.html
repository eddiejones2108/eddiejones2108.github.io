Linear Logic

Suppose apples cost a £1 If you have £1 you can have an apple and so we might be tempted to write £1 ⇒ 🍏 But from this we can derive £1 ⇒ 🍏 ⋀ 🍏, so it seems there is a buy one get one free offer on?

Linear logic acts as an accountant for greengroceers by considering resources rather than true. And unlike truth, resources can be used up. The proposition £1 ⊸ 🍏 specifically states that the pound is consumed in the production of the apple. It does not follow, therefore, that £1 ⊸ 🍏 ⊗ 🍏, where the ⊗ symbol clarrifies that we are actually talking about two apples.
